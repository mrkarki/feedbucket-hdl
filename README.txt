== Description ==

This plugin for enable the feedback features with the Team of developer and project managers.

At a glance, this plugin adds the following:

* Administrators can select the dispay type for all users or admin only.
* Administrators can allow users to change feedbucket script.
* When allowed, the users can choose front and beckend enable.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `feedbucket-hdl.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the realted configuration in settings section
